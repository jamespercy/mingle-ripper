As an account admin, I want to see the actual status of my account user (suspended/locked etc) so that I know what's going on

Scope

  • User status needs to be shown seperately from password status

Validations/Changes

All the changes apply to

- All internal accounts (CERadmin, helpdesk etc) - Manage internal users, manage external users

- External accounts - Manage users

Search Users

  • Password status values (Expired, Locked out, Active) to be shown seperately under different drop down "Password status"
  • "Status" to now be renamed to "User status"
  • The new drop down should be searchable and should be exactly same as User status in terms of behavior
  • Search results to now have another column to show Password status

User details

  • "Status" to now be renamed to "User status"
  • A new attribute to show "Password status"
